# what-the-ken-all

https://katai5plate.github.io/what-the-ken-all/

## Usage

### CSV を JSON 化

1. [ここ](http://zipcloud.ibsnet.co.jp/) から `加工済バージョン` の `x-ken-all.csv` を入手し、ルートに置く
2. `node xKenAllToJSON` を実行
3. `ken.json` が生成される

### JSON から地名リスト作成

- `node kenJSONToList` を実行する

### Docs 生成

- `npm run deploy` を実行する
